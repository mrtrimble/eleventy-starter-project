# Vickie's Practice site

Vickie is using this repository to practice the basics of using git, markdown, and building sites with a static site generator. 

For this project, Vickie will be learning:
- [X] Setting up a local dev environment
- [X] Forking repositories
- [X] Cloning repos to the local dev environment
- [X] Creating branches
- [X] Making changes to the repository
- [X] Updating content with markdown
- [X] Committing changes
- [X] Pushing changes to GitHub
- [X] Creating Pull Requests

## About this site
- Built using 11ty
- Based on the completed project from [Level-up With Modern CSS Workshop](#level-up-with-modern-css-workshop)

## Development Scripts

**`npm start`**

> Run 11ty with hot reload at localhost:8080, including reload based on Sass changes

**`npm run build`**

> Production build includes minified, autoprefixed CSS and site pages ready to deploy on an external host.
